# Bud Studio Desktop App Icons

## Required Icons

Place your app icons in this directory with the following names:

- **icon.png** - Base icon (1024x1024 recommended, PNG with transparency)
- **icon.icns** - macOS icon (auto-generated from icon.png if using electron-builder)
- **icon.ico** - Windows icon (auto-generated from icon.png if using electron-builder)

## Icon Requirements

### Source Image (icon.png)
- **Size**: 1024x1024 pixels minimum
- **Format**: PNG with transparency
- **Design**: The purple rounded square Bud Studio logo

### Platform-Specific

**macOS (.icns)**
- Contains multiple sizes: 16x16, 32x32, 64x64, 128x128, 256x256, 512x512, 1024x1024
- Automatically generated by electron-builder from icon.png

**Windows (.ico)**
- Contains: 16x16, 32x32, 48x48, 256x256
- Automatically generated by electron-builder from icon.png

**Linux (.png)**
- Uses icon.png directly

## Auto-Generation

When you run `npm run package`, electron-builder will automatically:
1. Use icon.png as the source
2. Generate icon.icns for macOS
3. Generate icon.ico for Windows
4. Use icon.png for Linux

## Manual Generation (Optional)

If you want to manually create the icons:

### macOS (.icns)
```bash
# Using iconutil (macOS only)
mkdir icon.iconset
sips -z 16 16 icon.png --out icon.iconset/icon_16x16.png
sips -z 32 32 icon.png --out icon.iconset/icon_16x16@2x.png
sips -z 32 32 icon.png --out icon.iconset/icon_32x32.png
sips -z 64 64 icon.png --out icon.iconset/icon_32x32@2x.png
sips -z 128 128 icon.png --out icon.iconset/icon_128x128.png
sips -z 256 256 icon.png --out icon.iconset/icon_128x128@2x.png
sips -z 256 256 icon.png --out icon.iconset/icon_256x256.png
sips -z 512 512 icon.png --out icon.iconset/icon_256x256@2x.png
sips -z 512 512 icon.png --out icon.iconset/icon_512x512.png
sips -z 1024 1024 icon.png --out icon.iconset/icon_512x512@2x.png
iconutil -c icns icon.iconset
```

### Windows (.ico)
Use a tool like ImageMagick:
```bash
convert icon.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico
```
